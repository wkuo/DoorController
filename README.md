# DoorController

To control door lock/unlock, open/close


Pin Mapping (ESP8266 - SD Module)
D2 - CS 
D5 - SCK
D6 - MISO 
D7 - MOSI 
Vin - Vcc 
GND - GND


Pin Mapping (ESP8266 - Shift Register 4595 )
D6 (GPIO12) - SER    
D7 (GPIO13) - RCLK  
D8 (GPIO15) - SRCLK
3v3 - Vcc & SRCLR (bar) 
GND - GND & OE (bar)
